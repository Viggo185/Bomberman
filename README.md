# Bomberman
This game is a recreation of the classic Bomberman with a focus on local multiplayer. Players compete on the same screen by placing bombs, avoiding explosions, and trying to be the last one standing. The game features animated sprites and items that affect gameplay.

Key Features
* ### Movement:
Players can move freely in four directions across the map with responsive controls.
* ### Bomb Placement:
Players can place bombs that explode after a short delay, destroying obstacles and damaging players.
* ### Spawn System:
Players spawn at predefined positions on the map at the start of each match.
* ### Destruction of Bricks:
Breakable bricks can be destroyed by bomb explosions, sometimes revealing hidden items or power-ups.
* ### Player Death System:
Players die when caught in a bomb explosion, leading to elimination from the current round.
* ### Items / Power-ups:
Items can drop from destroyed bricks and give players advantages, such as additional bombs or increased explosion range.
* ### Collision System:
Handles collisions between players, walls, bombs, and explosions.
* ### Animated Sprites:
Characters and objects are animated to create a more dynamic and engaging experience.
* ### Local Multiplayer:
Multiple players can play together on the same device/keyboard.
